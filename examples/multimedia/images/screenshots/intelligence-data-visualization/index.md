## Overview

This demo walkthrough showcases IntellyWeave's multiple visualization methods for exploring intelligence data, including text responses, tables, documents, threads, charts, network diagrams, maps, courthouse debates, and intelligence agent analysis.

## Summary

This walkthrough covered IntellyWeave's comprehensive visualization toolkit:

| Category | Features |
|----------|----------|
| **Text & Documents** | Text Response, Table, Document, Single Message, Thread |
| **Data Analysis** | Aggregation, Entity Statistics, Source Code |
| **Charts** | Bar Chart, Network Chart |
| **Geospatial** | Mapbox 3D Maps |
| **AI Analysis** | Courthouse Debate, Intelligence Agent |

Each visualization method provides a different lens for analyzing intelligence data, from structured tables to interactive network diagrams and adversarial AI debates.
