# Support

Welcome to IntellyWeave! Here's how to get help.

## Getting Help

### 1. Documentation

Start with the [README](README.md) and [docs/](docs/) directory:

- Quick start guide
- Configuration options
- Development setup
- Troubleshooting

### 2. GitHub Discussions

For questions, ideas, and community discussion, use [GitHub Discussions](https://github.com/vericle/intellyweave/discussions).

**Good for:**

- How-to questions
- Feature discussions
- Sharing use cases
- General Q&A

### 3. GitHub Issues

Use [GitHub Issues](https://github.com/vericle/intellyweave/issues) for:

- Bug reports
- Feature requests

**Not for:** General questions (use Discussions instead).

## Response Times

This project is maintained by one person. Response times vary based on:

- Issue severity
- Maintainer availability
- Complexity of the problem

Please be patient. This is an open-source project maintained in spare time.

See [SECURITY.md](SECURITY.md) for responsible disclosure instructions.
